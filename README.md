# 💰 Discount Disaster Analysis

## 🎯 Problem Statement
CEO ne Q4 mein heavy discounts diye sales badhane ke liye. 
Revenue toh dikha, lekin **profit gayab ho gaya**. 
Kya discount strategy actually kaam kar rahi thi?

## 🛠️ Tools Used
- **Google Colab** (Python environment)
- **Pandas** (Data manipulation)
- **Matplotlib** (Visualization)
- **AI Assistance** (ChatGPT/Claude for code generation)

## 📊 Key Findings
| Discount | Revenue Impact | Profit Impact |
|----------|---------------|---------------|
| 0% | Baseline | Baseline |
| 10% | +15% | +12% ✅ **Sweet Spot** |
| 20% | +25% | +5% |
| 30% | +35% | -20% ⚠️ |
| 50% | +40% | -60% ❌ **Disaster** |

## 💡 Business Recommendations
1. **Never exceed 20% discount** on electronics
2. **10% discount** maximizes total profit
3. **Volume play** works only if margin &gt;40%
